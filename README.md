# 📒 Agenda de Contatos em Python

Projeto desenvolvido em Python com o objetivo de praticar lógica de programação, manipulação de listas e dicionários, validação de dados, funções e estruturas de repetição.

---

# 🚀 Funcionalidades

O sistema permite:

- ✅ Adicionar contatos
- ✅ Alterar contatos
- ✅ Remover contatos
- ✅ Listar contatos cadastrados
- ✅ Validação de dados de entrada
- ✅ Menu interativo no terminal

---

# 🛠️ Tecnologias Utilizadas

- Python 3

---

# 📚 Conceitos Aplicados

Durante o desenvolvimento deste projeto foram utilizados conceitos como:

- Strings
- Listas
- Dicionários
- Funções
- Loops (`while` e `for`)
- Estruturas condicionais (`if`, `elif`, `else`)
- Métodos de strings
- Tratamento de entradas do usuário
- Validação de dados
- Iteração com `enumerate`
- Manipulação de listas e dicionários

---

# 📋 Estrutura do Contato

Cada contato é armazenado como um dicionário dentro de uma lista.

Exemplo:

```python
contato = {
    'nome': 'João Silva',
    'telefone': '21999999999'
}
```

# ▶️ Como Executar

1. Certifique-se de ter o Python instalado.
2. Clone este repositório ou baixe o arquivo.

# ✅ Validações Implementadas

O sistema possui validações para:

- Nome contendo apenas letras
- Telefone contendo apenas números
- Telefone com exatamente 11 dígitos
- Verificação de contatos existentes
- Lista vazia
